### 1. Fork and Clone

```bash
git clone https://github.com/yourusername/wasmrun.git
cd wasmrun
```

### 2. Development Setup

```bash
# Install Just task runner
cargo install just

# Build the project
just build

# Run tests
just test

# Format and lint
just format
just lint
```

### 3. Make Your Changes

Follow the guidelines in our [Contributing Guide](https://wasmrun.readthedocs.io/en/latest/docs/development/contributing).

### 4. Submit a Pull Request

- Create a branch with a descriptive name
- Make your changes following our code style
- Add tests for new functionality
- Update documentation if needed
- Submit a PR with a clear description
